# Graphics-Exercise

Note! Η εκφώνηση ζητάει να δοθεί και το εκτελέσιμο πρόγραμμα μαζί με τα                                    
πηγαία αρχεία. Κατά την μεταφορά όμως των αρχείων σε άλλο σύστημα,
χάνουν το δικαίωμα εκτέλεσης. Γιά να γίνει εκτέλεση, είτε εκτελέστε make 
clean για να αφαιρεθεί το εκτελέσιμο αρχείο και μετά make run, είτε τρέξτε
chmod +x build/planets” για να δοθεί το δικαίωμα εκτέλεσης.

makefile : 
Για compilation, make.
Για να τρέξει το πρόγραμμα, make run. 

Είναι αναγκαίες οι βιβλιοθικες GLFW3 και CGLM :
sudo apt install libglfw3-dev
sudo apt install libcglm-dev
